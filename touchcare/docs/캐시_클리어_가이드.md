# 캐시 클리어 가이드

## 🔧 Metro Bundler 캐시 완전히 클리어하기

### 1단계: 개발 서버 완전히 종료
```bash
# 터미널에서 Ctrl+C로 서버 중지
```

### 2단계: 캐시 클리어 및 재시작
```bash
cd touchcare-mobile

# Metro bundler 캐시 클리어
npx expo start --clear

# 또는 더 강력한 클리어
rm -rf node_modules/.cache
npx expo start --clear
```

### 3단계: Expo Go 앱에서 완전히 재시작
1. **Expo Go 앱 완전히 종료** (백그라운드에서도 종료)
2. **앱 삭제 후 재설치** (필요한 경우)
3. **다시 QR 코드 스캔**

### 4단계: Watchman 캐시 클리어 (Mac/Linux)
```bash
watchman watch-del-all
```

### 5단계: node_modules 재설치 (최후의 수단)
```bash
cd touchcare-mobile
rm -rf node_modules
npm install
npx expo start --clear
```

## 🔍 확인 방법

터미널에서 `🚀 App.tsx 로드됨` 메시지가 보이면 정상입니다.
만약 이 메시지가 안 보이면 여전히 캐시된 파일을 사용하고 있는 것입니다.

## ⚠️ 주의사항

- `App.js` 파일이 있는지 확인 (있으면 삭제)
- `index.js`가 `./App`을 import하는지 확인 (TypeScript는 자동으로 `.tsx`를 찾음)
