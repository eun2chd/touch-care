# TouchCare 최종 컬러 컨셉

## 🎨 선택된 컨셉: 클린 메디컬 + 뉴트럴 마인드 하이브리드

전문성과 감성 케어의 완벽한 조합으로, 클린 메디컬의 명료함과 뉴트럴 마인드의 따뜻함을 결합한 컨셉입니다.

---

## 📋 색상 팔레트

### 메인 컬러

#### 1. Indigo (#303F9F)
- **출처**: 클린 메디컬
- **용도**: 
  - 메인 버튼
  - 제목 텍스트
  - 헤더 배경
  - 핵심 인디케이터
- **의미**: 전문성, 신뢰감, 명료함

#### 2. Sage Green (#81C784)
- **출처**: 뉴트럴 마인드
- **용도**:
  - 보조 버튼
  - 안정 상태 표시
  - 긍정적 피드백
- **의미**: 심리적 안정, 회복, 평온함

### 서브 컬러

#### 3. Slate Gray (#78909C)
- **출처**: 클린 메디컬
- **용도**:
  - 서브 텍스트
  - 그래프 가이드라인
  - 데이터 분석 표시
- **의미**: 차분함, 전문성

#### 4. Ice Blue (#E3F2FD)
- **출처**: 클린 메디컬
- **용도**:
  - 선택된 영역 배경
  - 강조 영역
  - 카드 테두리
- **의미**: 깔끔함, 명료함

### 강조 컬러

#### 5. Muted Coral (#EF9A9A)
- **출처**: 뉴트럴 마인드
- **용도**:
  - 주의/경고 상태
  - 충전 중 LED (Breathing 효과)
  - 감성적 강조
- **의미**: 따뜻함, 주의, 케어

### 배경 & 텍스트

#### 6. Pure White (#FFFFFF)
- **출처**: 클린 메디컬
- **용도**:
  - 전체 배경색
  - 카드 배경
  - 여백
- **의미**: 깔끔함, 명료함, 최소한의 인지 부하

#### 7. Charcoal (#37474F)
- **출처**: 뉴트럴 마인드
- **용도**:
  - 본문 텍스트
  - 가독성 높은 텍스트
- **의미**: 안정감, 가독성

#### 8. Warm Beige (#FDF5E6) - 옵션
- **출처**: 뉴트럴 마인드
- **용도**:
  - 특정 카드 배경 (선택적)
  - 따뜻함이 필요한 영역
- **의미**: 따뜻함, 편안함

## 🎯 색상 사용 가이드

### 버튼
- **Primary**: Indigo (#303F9F) - 주요 액션
- **Secondary**: Sage Green (#81C784) - 보조 액션
- **Accent**: Muted Coral (#EF9A9A) - 강조 액션
- **Tertiary**: Ice Blue 배경 + Indigo 텍스트 - 선택 영역

### 상태 표시
- **안정 상태**: Sage Green (#81C784)
- **주의 필요**: Muted Coral (#EF9A9A)
- **데이터 분석**: Slate Gray (#78909C)
- **활성 상태**: Indigo (#303F9F)

### 배경
- **전체 배경**: Pure White (#FFFFFF)
- **카드 배경**: Pure White (#FFFFFF) + Ice Blue 테두리
- **선택 영역**: Ice Blue (#E3F2FD)

### 텍스트
- **제목**: Indigo (#303F9F)
- **본문**: Charcoal (#37474F) 또는 Gray (#666)
- **서브 텍스트**: Slate Gray (#78909C)

## 💡 디자인 원칙

1. **명료함**: 흰색 배경으로 깔끔한 UI
2. **균형**: 전문성(Indigo)과 감성(Sage Green, Muted Coral)의 조화
3. **가독성**: 충분한 대비와 명확한 계층 구조
4. **일관성**: 하드웨어 LED 색상과 UI 색상 통일
