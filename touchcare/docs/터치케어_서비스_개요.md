# 터치케어(TouchCare) 서비스 개요

## 📱 서비스 소개

**터치케어**는 정신건강 예방을 위한 **SaMD (Software as a Medical Device)** 플랫폼입니다.

---

## 🏗️ 시스템 아키텍처

### 1. 하드웨어 (IoT 기기)
- **기반**: ESP32 기반 IoT 기기
- **기능**: 사용자 터치 감지
- **통신**: BLE 5.0 (Bluetooth Low Energy)을 통해 앱으로 데이터 전송

### 2. 모바일 앱
- **역할**: IoT 기기와 통신하여 터치 데이터 수신
- **기술**: React Native + Expo (목업 개발)

### 3. 백엔드 & AI
- **데이터베이스**: Supabase
- **AI 엔진**: 수집된 로그를 분석하여 리포트 생성
- **보안**: 
  - AES-256 암호화
  - RLS (Row Level Security)로 보안 강화

### 4. 특화 기능
- **파트너 간 비언어적 연결**: 터치 데이터를 통한 감성적 소통 기능 제공

---

## 🔐 보안 & 프라이버시

- **AES-256 암호화**: 모든 데이터 전송 및 저장 시 암호화
- **RLS (Row Level Security)**: 데이터베이스 레벨에서 접근 제어
- **의료기기 등급**: SaMD로서의 보안 표준 준수

---

## 📊 주요 기능

1. **터치 데이터 수집**: IoT 기기를 통한 실시간 터치 감지
2. **데이터 분석**: AI 엔진을 통한 패턴 분석
3. **리포트 생성**: 사용자 맞춤형 정신건강 리포트 제공
4. **파트너 연결**: 비언어적 소통을 통한 감성적 연결

---

## 🎯 목표

정신건강 예방을 위한 **데이터 기반 인사이트**와 **감성적 케어**를 결합한 통합 솔루션 제공
